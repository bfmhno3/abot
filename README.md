# A.B.O.T (Agricultural & Botanical Observation Tracker)

> **⚠️ 状态：已归档 (Archived)**
>
> 本仓库仅用于存放早期通识选修课程的原始文本记录与实验废稿。因数据采集设备误差及部分植株样本枯萎，该观察计划已于上学期中止。请勿将此仓库内的数据用于任何正式的学术报告或论文引用。

## 1. 项目简介

A.B.O.T 是一个用于记录特定试验田区域内，常见阔叶杂草在不同微量元素胁迫下的表型特征变化的个人存档。最初的设想是建立一个长期的监测日志，但由于手动测量的繁琐性以及冬季气温骤降导致的样本死亡，该记录已失去连续性。

本分支仅包含纯文本的初步观察计划和一些杂乱的数据表单。

## 2. 观察方法与指标

在为期三周的有效观察期内，主要采用游标卡尺和比色卡进行人工采样，主要记录以下毫无技术含量的枯燥指标：

- **叶片伸展度 (Leaf Span):** 测量主叶脉的绝对长度（单位：mm），误差 ±2mm。
- **叶绿素相对水平:** 基于视觉比对，分为浅绿、灰绿、深绿三个粗略等级。
- **土壤表面湿度 (Surface Moisture):** 纯主观感受评估（干燥/微润/湿涝）。

## 4. 为什么放弃？

1. **不可控因素过多：** 露天观察极易受到天气、路人以及动物的干扰。
2. **数据无意义：** 缺乏精确的传感器介入，单靠肉眼和游标卡尺记录的数据无法进行任何有价值的统计学分析。
3. **精力有限：** 期中考试临近，已无暇顾及此等繁琐且学分占比较低的选修课任务。
